# fox-reminder ダウンロード

ふわっち（Whowatch）のライブ配信コメントをリアルタイム表示する **Windows デスクトップアプリ** です。

## ダウンロード

最新版は **[Releases](https://github.com/a605043/fox-reminder/releases)** から取得してください。

| ファイル | 説明 |
|----------|------|
| `fox-reminder Setup x.x.x.exe` | インストーラー版（推奨） |
| `fox-reminder x.x.x.exe` | ポータブル版（インストール不要） |

## 使い方（概要）

1. 上記 Releases から exe をダウンロードして起動する
2. 設定画面の **配信の指定** に、ふわっちの視聴 URL（`/viewer/{id}` / `/archives/{id}`）を入力する
3. **接続** を押してコメント一覧を表示する

## 注意事項

- 本アプリはコード署名されていません。初回起動時に Windows SmartScreen の警告が表示される場合があります。「詳細情報」→「実行」で起動できます
- ふわっち API は**非公式**です。仕様変更で動作しなくなる可能性があります
- 配信者・視聴者の利用規約を遵守してください
- 本アプリはコメントの**閲覧のみ**を行い、投稿機能はありません
